## Simpliflow - Login & Registration Page

### Features:
A responsive login/registration page built with pure HTML and CSS Features

Split Layout: 70% brand section (left) and 30% form section (right)
Tab Switching: Toggle between Login and Register forms
Fully Responsive: Adapts to mobile, tablet, and desktop
No JavaScript: Pure CSS implementation

- Structure Left Section (70%)

    Logo with circular styling
    Welcome message and description
.

- Right Section (30%)

    Login/Register tabs
    Form inputs and buttons
    Helper links (Forgot password etc,.)

### Objective

To practice HTML structure, CSS layouts, responsiveness.
