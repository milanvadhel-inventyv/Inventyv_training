## Modules Overview

### 01_LOGIC_BUILDING
This module focuses on strengthening **logical thinking and problem-solving skills**.  
Concepts are approached using:
- Step-by-step logic
- Flowcharts
- Structured reasoning techniques  

---

### 02_HTML_CSS_LOGINPAGE
A **responsive login and registration page** built using **pure HTML and CSS**.


---

### 03_SQL_QUERY
A **hands-on DBMS assignment** involing multiple sql based query on a **movie database schema**.

---

### 04_JAVASCRIPT_TASK

**Topics include:**
- Arrays and object manipulation
- Function execution flow
- Print Numerical Patterns with Constraints
- Promises and asynchronous handling
- Usage of Symbol
---

### 05_Exercism_Javascript

This folder includes JavaScript Learn Mode exercises completed on Exercism to improve JavaScript fundamentals and logic building,
By Solving Various Problem.

---

### 06_ANGULAR
This folder includes Practises of  Angualr Assignement where different Angular App/project Created for understanding the Angular Concept with Practical Implementation.
